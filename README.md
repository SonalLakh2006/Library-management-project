# 📚 Library Management System (C++)

## 📖 Project Overview
The **Library Management System** is a console-based application developed in **C++**. It helps manage library records by allowing users to add, search, issue, and return books. The project demonstrates the use of **Object-Oriented Programming (OOP)** concepts and **File Handling** for storing data.

---

## 🎯 Aim
To develop a simple Library Management System using C++ that helps manage book records, issue and return books, and search books using object-oriented programming and file handling.

---

## ✨ Features
- ➕ Add new books
- 📋 Display all available books
- 🔍 Search books by title
- 👤 Search books by author
- 📖 Issue books to users
- 🔄 Return issued books
- 💾 Store book records using file handling
- 🏛️ Object-Oriented Programming using classes

---

## 🛠️ Technologies Used
- **Programming Language:** C++
- **Concepts:** Object-Oriented Programming (OOP)
- **File Handling:** File I/O
- **Platform:** Console Application

---

## 📂 Project Scope
This project is suitable for small libraries to maintain book records efficiently. It helps keep track of available and issued books, reduces manual work, and provides a simple way to search, issue, and return books.
